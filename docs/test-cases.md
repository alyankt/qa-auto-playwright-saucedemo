# Тест-кейсы

## TC-001 Вход с валидными данными

Предусловие:
Пользователь находится на странице логина.

Шаги:
1. Ввести логин 'standard_user'.
2. Ввести пароль.
3. Нажать на кнопку 'Login'.

Ожидаемый результат:
Открывается страница товаров.

## TC-012 Оформление заказа без имени

Предусловие:
Пользователь добавил товар в корзину и открыл форму оформления заказа.

Шаги:
1. Оставить поле 'First Name' пустым.
2. Заполнить поле 'Last Name'.
3. Заполнить поле 'Postal Code'.
4. Нажать на кнопку 'Continue'.

Ожидаемый результат:
Появляется ошибка, что поле 'First Name' обязательно.

## TC-015 Успешное оформление заказа

Предусловие:
Пользователь авторизован и добавил товар в корзину.

Шаги:
1. Открыть корзину.
2. Нажать на кнопку 'Checkout'.
3. Заполнить данные покупателя.
4. Нажать на кнопку 'Continue'.
5. Проверить товар на странице обзора.
6. Нажать на кнопку 'Finish'.

Ожидаемый результат:
Заказ успешно оформлен.
