# QA Auto Playwright Saucedemo

Небольшой учебный проект по автоматизации UI-тестов. Делала его, чтобы потренироваться в Playwright, TypeScript, Page Object и запуске тестов через GitHub Actions.

Тестируемый сайт: https://www.saucedemo.com/

## Стек

- TypeScript
- Playwright
- Page Object Model
- GitHub Actions
- Git

## Что проверяется

- вход с валидными данными
- ошибки при пустом логине или пароле
- ошибка при неверном пароле
- вход заблокированного пользователя
- отображение списка товаров
- сортировка товаров по цене
- открытие карточки товара
- добавление товара в корзину
- удаление товара из корзины
- ошибки в форме оформления заказа
- успешное оформление заказа

## Что где лежит

- `auth.spec.ts` – логин и ошибки входа
- `inventory.spec.ts` – список товаров, сортировка, карточка товара
- `cart.spec.ts` – добавление, отображение и удаление товара из корзины
- `checkout.spec.ts` – оформление заказа и проверки формы

## Структура проекта

```text
tests/      - тесты
pages/      - Page Object классы
fixtures/   - тестовые данные
docs/       - тестовая документация
```

## Установка

```bash
npm install
npx playwright install
```

## Запуск тестов

```bash
npm test
```

## Запуск в видимом браузере

```bash
npm run test:headed
```

## HTML-отчёт

```bash
npm run report
```

## CI

После загрузки изменений на GitHub тесты запускаются автоматически.
